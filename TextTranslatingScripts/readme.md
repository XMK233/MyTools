This is a paper reading script for researchers who have hard time reading English literature. 这是一个可以翻译英文文献的脚本，是为英文文献所苦之人的好帮手。

运行md.py. 用一些比较强大的文本编辑器，诸如vscode, sublime之类，打开`.pts`文件。可以用`run_md_.bat`脚本同时做到上述操作。

然后把英语复制到source.pts当中保存。根据逻辑，首先会将所有的换行符替换为空格/ 有一些句号比如e.g.这样的会被替换为其他符号。

片刻，就可以看到target.pts里面出现了翻译。注意翻译是按照句号划分之后逐句翻译的。

尤为适合于翻译大量文字的文章，比如社会学，文学等。有一些英语的习惯表达法，如果没有翻译的话，你根本无法理解是什么意思，就跟看文言文似的。
